# 🇧🇷 Blog - Realidade do Brasil

Um blog moderno desenvolvido com HTML, CSS e JavaScript, focado em apresentar conteúdos sobre a realidade social, política e cultural do Brasil.

## 🌐 Demonstração

🔗 Acesse o projeto:

---

## 💡 Sobre o projeto

Este projeto foi criado com o objetivo de desenvolver um blog informativo com foco na realidade brasileira, abordando temas como:

- Desigualdade social
- Política e corrupção
- Cultura e identidade nacional
- Tecnologia no Brasil

O layout foi pensado para ser limpo, moderno e inspirado em portais de notícias profissionais 📰

---

## 🎯 Objetivos

- 📌 Criar um blog responsivo
- 💼 Demonstrar habilidades em Front-End
- 🇧🇷 Abordar temas relevantes do Brasil
- 🎨 Trabalhar design e experiência do usuário

## 🧠 Funcionalidades
